### Chrome (MAC)

**Function** | **Shortcut**
-------- | --------
change between tabs |`⌘` + `⌥` + `arrow` 
open inspector |`⌘` + `⇧` + `i` 
reload page |`⌘` + `r` 
focus on navigation bar |`⌘` + `l` 
open new window |`⌘` + `n` 
open new tab |`⌘` + `t` 
close current tab |`⌘` + `w` 
open closed tab |`⌘` + `⇧` + `t` 
close chrome |`⌘` + `q`
